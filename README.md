# 🧠 Braion - AI Team Name Generator

An intelligent team name generator that creates context-aware, meaningful, and creative names for hackathons, startups, clubs, and competitions.

## ✨ Features

- **Dual Generation Mode**
  - 💡 Meaningful Names: Based on your concepts and domain
  - ✨ Creative Names: Cool, catchy random combinations

- **Context-Aware Intelligence**
  - Multi-domain support (AI, IoT, Healthcare, etc.)
  - Tone customization (Professional, Cool, Funny, Aggressive, Minimal)
  - Project description integration
  - Custom naming instructions

- **Interactive Chatbot UI**
  - Natural conversation flow
  - Real-time name generation
  - Quick action buttons

## 🛠️ Tech Stack

**Backend:** FastAPI, NLTK, WordNet, Python  
**Frontend:** React, Vite, Lucide Icons

## 🚀 Installation

### Backend Setup
\\\ash
pip install -r requirements.txt
python -m uvicorn backend.main:app --reload
\\\

### Frontend Setup
\\\ash
cd frontend
npm install
npm run dev
\\\

Visit \http://localhost:5173\ to use Braion!
